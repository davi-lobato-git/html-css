Listas

Listas Ordenadas

Usando a tag <ol> para criação da lista (podendo formatar o tipo de ordem com o parâmetro type e o parâmetro de início start) e a tag <li> para cada item dela.

  1. Acordar
  2. Desligar os LEDs
  3. Tomar Café
  4. Escovar os Dentes

Listas Não Ordenadas

Para listas sem ordem lógica, usamos a tag <ul> (também podendo alterar o símbolo de ordem com o parâmetro type).

  • Pão
  • Leite
  • Arroz
  • Feijão

Juntando Listas

Utilização de mais de uma tag <ol> ou <ul> dentro de outra:

  • Filmes
    • Terror
      1. O Grito
        • Japonês
        • Americano
      2. O Chamado
      3. Espíritos
    • Thriller
      1. Seven
      2. Zodíaco
      3. Oldboy
