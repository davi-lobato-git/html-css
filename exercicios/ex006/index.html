Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica);

Nessa frase temos um termo em destaque usando a tag <strong> (semântica).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica);

Nessa frase temos um termo em itálico usando a tag <em> (semântica).

Texto Marcado

Nessa frase temos um termo marcado usando a tag <mark>.

Texto Grande e Pequeno

Nessa frase temos um termo grande usando a tag <big> (sem sentido e obsoleto) e um termo pequeno usando a tag <small>.

Texto Deletado

Nessa frase temos um termo que deve ser lido, mas não considerado, usando a tag <del>.

Texto Inserido

Nessa frase temos um termo que indica ênfase e inclusão posterior usando a tag <ins>. Não confundir com a tag de sublinhado <u> que é não semântica.

Texto Sobrescrito

Para inserir algo como x2 usando a tag <sup>.

Texto Subescrito

Para inserir algo como H2O usando a tag <sub>.
